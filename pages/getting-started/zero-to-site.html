不会代码怎么用AI搭网站？保姆级7步教程

📅 2026-05-15 ⏱ 15分钟 入门
教程 零基础 建站

这篇教程就是写给完全不懂编程的你。跟着这7步走，你可以在一个下午之内，用AI做出一个属于自己的网站。不需要写一行代码，不需要花一分钱。

你需要什么？其实就两样东西

在开始之前，清点一下你的装备：

  • 一台能上网的电脑 — 台式机、笔记本都可以。Windows或Mac都行。
  • 一个浏览器 — Chrome、Edge、Safari都可以。推荐用Chrome。

就这些。不需要装任何软件，不需要懂任何编程知识。你只需要会打字、会用鼠标。

现在我们来一步步做。

第一步：选工具——推荐ChatGPT

能把你的想法变成网站，全靠AI工具。目前最适合新手的AI工具有两个：

  • ChatGPT（chat.openai.com）— 免费，操作简单，中文支持好
  • Claude（claude.ai）— 也免费，写代码能力很强

如果你是第一次用，推荐先选ChatGPT。注册只需要一个邮箱或Google账号，完全免费。打开网站就能直接开始聊天。

小提示： 这两个工具都是"对话式"的，就像你和一个很聪明的朋友聊天一样。你打字问它问题，它就会回答你。不需要学任何操作。

第二步：告诉AI你想要什么

很多人第一步就卡住了——不知道该怎么跟AI说。其实很简单，你就把它当成一个什么都会做的设计师兼程序员。

跟AI说话的关键是：说清楚你想要什么类型的网站，用来做什么。不用说得太复杂，也不用担心说错。

这里给你一个可以直接复制使用的模板：

请帮我生成一个个人简历网页，风格简洁现代。

要求包含以下几个部分：
1. 最上面是我的姓名和一句话介绍
2. 我的工作经历（列出2-3段工作）
3. 我做过的项目展示
4. 底部是我的联系方式

颜色用蓝色系，整体要干净清爽。电脑和手机上都能正常显示。
请直接给我完整的HTML代码，我复制保存就能用。

这个模板适用于大多数个人网站。如果你想要其他类型，比如作品集、公司介绍页、活动宣传页，把"个人简历"换成你想要的类型就行。

第三步：AI帮你生成网站

把上面的提示词复制到ChatGPT里，按回车发送。等几秒钟，AI就会给你一段完整的HTML代码。

它看起来会是这样的一大段文字，有尖括号和英文单词——别被它吓到，你不需要理解它。你只需要做一件事：

  1. 把AI回复的代码全部选中，复制
  2. 打开电脑上的"记事本"（Windows）或"文本编辑"（Mac）
  3. 把代码粘贴进去
  4. 点击"文件" → "另存为"
  5. 文件名写 index.html（注意后缀是.html，不是.txt）
  6. 保存到桌面上

保存之后，双击这个文件，它就会在你的浏览器里打开——一个真正的网站！

常见问题： 如果保存后图标不是浏览器样式，说明后缀名不对。确保文件名是 index.html 而不是 index.html.txt。如果看不清楚，可以问AI"我的文件怎么保存成HTML"，它会一步步教你。

如果你想做更复杂的网站，比如带图片的展示页面，可以用这个提示：

请帮我做一个"我的作品集"网站，用来展示我的设计作品。

要求：
1. 顶部导航栏，有"首页""作品""关于我""联系"四个菜单
2. 首页有一个大背景图区域，上面写我的名字和一句话
3. 作品页面以网格形式展示6个作品（每个用占位图就行）
4. 关于我页面写一段个人介绍
5. 联系页面放一个简单的联系表单
6. 整个网站风格要时尚有设计感，用深色背景
7. 所有页面在一个HTML文件里，点击导航切换

请直接给我完整的HTML代码。

第四步：预览和修改——不满意就继续问

你可能会觉得第一版不太满意。颜色不好看、排版不对、少了一些内容——这些都很正常。

修改的方法特别简单：直接告诉AI你要改什么。

比如：

刚才的代码我试了，整体不错，但有几个地方想改：
1. 蓝色太深了，换成浅蓝色
2. 在"工作经历"下面加一个"教育背景"部分
3. 字体调大一点
4. 底部的"联系方式"改成靠右对齐

请给我更新后的完整HTML代码。

AI会重新生成一份完整的代码。你重复之前的操作——复制、粘贴到记事本、保存、双击打开。就这么简单。

关键技巧：

  • 一次只说一个修改方向 — 不要一次说10个改动，容易把AI搞晕
  • 用"更"字来调整 — "字体更大一点""间距更宽一点""颜色更亮一点"
  • 不满意就重新来 — 如果改了好几轮还是不满意，可以重新换个提示词从头开始

第五步：免费部署上线——让别人也能访问

现在你的网站还在你自己的电脑上，别人看不到。要把网站放到互联网上，我们用一个完全免费的工具——Vercel。

跟着这几步操作：

  1. 注册GitHub账号 — 打开 github.com，点击"Sign up"，用邮箱注册。免费的。
  2. 创建仓库 — 登录后点击右上角的"+"号 → "New repository"，名字随意，选"Public"，点"Create"
  3. 上传你的文件 — 在新页面里点击"uploading an existing file"，把你保存的 index.html 拖进去，点击"Commit changes"
  4. 打开Vercel — 访问 vercel.com，点击"Sign Up"，选择"Continue with GitHub"，授权登录
  5. 部署 — 点"Add New Project"，找到你刚创建的GitHub仓库，点"Import"，然后点"Deploy"

等一两分钟，Vercel会给你一个网址，类似 你的项目名.vercel.app。把这个网址发给任何人，他们都能打开你的网站了。

完全免费吗？ 是的，Vercel的免费套餐对于个人网站完全够用。每月有100GB的流量，即使每天几百人访问也没问题。不需要绑定银行卡。

第六步：绑定一个自己的域名（可选但推荐）

Vercel给的网址虽然能用，但不好记。如果你想拥有一个像 你的名字.com 这样的网址，可以自己买一个域名。

买域名的地方：

  • 阿里云（aliyun.com）— 国内用户首选，支持支付宝
  • Namesilo（namesilo.com）— 国外平台，价格便宜
  • GoDaddy（godaddy.com）— 全球最大的域名注册商

一个 .com 域名一年大概50-80块钱。买好之后，把域名指向Vercel——这个过程在Vercel的设置里叫"Add Domain"，输入你的域名，它会告诉你需要在购买域名的网站做啥设置，跟着做就行了。

如果你暂时不想花钱买域名，用Vercel的免费网址也完全OK。等网站做出效果了再升级也不迟。

第七步：分享你的网站——大功告成！

网站上线了！现在你要做的事情就是：

  • 把网址发到朋友圈
  • 发到家庭群、同学群
  • 印在名片上
  • 放在你的简历里

有人访问你的网站了，恭喜你！你已经成功从"完全不懂"变成了"有一个自己的网站"的人。

接下来你可以：

  • 继续用AI添加新页面
  • 把网站内容更新得更丰富
