# AI编程指南

用AI写代码的完全攻略 — 从入门到精通的AI编程中文教程站

访问网站：https://ai-coding-handbook.vercel.app

---

## 关于本站

这是一个专注于AI编程攻略的中文内容站。涵盖 Claude Code、Cursor、GitHub Copilot、ChatGPT 等主流AI编程工具，从入门到实战的全方位教程。

适合人群：
- 零基础小白 — 想了解AI编程，从零开始
- 初级开发者 — 想用AI提升开发效率
- 中级开发者 — 想精进AI工作流
- 技术管理者 — 想评估团队AI引入方案

## 内容分类

| 分类 | 说明 | 文章数 |

## 技术栈

- HTML + Tailwind CSS CDN
- Noto Sans SC 字体
- highlight.js 代码高亮
- Fuse.js 客户端搜索
- Vercel 部署

## 特性

- 暗色模式（手动切换 + 跟随系统）
- 全文搜索（Ctrl+K）
- 响应式设计
- 文章目录自动生成
- 代码一键复制
- 阅读进度条

## 本地运行

```
git clone https://github.com/tutu230201/ai-coding-handbook.git
cd ai-coding-handbook
npx http-server -p 8080
```

打开浏览器访问 http://localhost:8080
