## 📖 关于本站

这是一个专注于 **AI编程攻略** 的中文内容站。涵盖 Claude Code、Cursor、GitHub Copilot、ChatGPT 等主流AI编程工具，从入门到实战的全方位教程。

**适合人群：**
- 🟢 **零基础小白** — 想了解AI编程，从零开始
- 🔵 **初级开发者** — 想用AI提升开发效率
- 🟡 **中级开发者** — 想精进AI工作流
- 🔴 **技术管理者** — 想评估团队AI引入方案

## 🛠️ 技术栈

- **HTML** — 纯静态多页面
- **CSS** — Tailwind CSS CDN（零构建）
- **字体** — Noto Sans SC（Google Fonts）
- **代码高亮** — highlight.js
- **搜索** — Fuse.js（客户端模糊搜索）
- **部署** — Vercel

## ✨ 特性

- 🌓 暗色模式 — 手动切换 + 跟随系统 + 持久化
- 🔍 全文搜索 — Ctrl+K 快捷搜索
- 📱 响应式设计 — 移动端/桌面端适配
- 📑 文章目录 — 自动生成 + 滚动高亮
- ⚡ 代码复制 — 一键复制代码块
- 📊 阅读进度 — 顶部进度条
